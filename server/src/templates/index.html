Tidy_Talk

Transform your verbal chatter into cleaned, structured text!




How to use

Load your audio file (.mp3, .wav), click `Process File`, and then wait patiently.

Your output file (.docx) will download automatically when ready.


Notes

It can handle long audio files (>10 minutes), though this will increase the wait time.

N.B. Maximum file size is 25mb. If your file size is above this, consider converting to a more compressed audio form (e.g., .mp3), or split into multiple files.
